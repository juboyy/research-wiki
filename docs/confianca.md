# Confiança: Cuidados com Idosos e PCDs

A terceira vertente do projeto AMIAU foca em tecnologias assistivas para **idosos** e **pessoas com deficiência (PCDs)**, promovendo autonomia, dignidade e qualidade de vida.

## O Desafio do Envelhecimento

Com o envelhecimento populacional global:
- **Isolamento social** — Companheiros animais reduzem solidão
- **Quedas** — Principal causa de hospitalização em idosos
- **Doenças neurodegenerativas** — Detecção precoce via padrões de vocalização

## Wearables para Idosos

### Coleira AMIAU Adaptada

| Sensor | Função para Idoso |
|--------|-------------------|
| Acelerômetro | Detecção de quedas, monitoramento de atividade |
| Temperatura | Detecção de febre/desidratação |
| Microfone | Comandos de voz, detecção de emergência |
| GPS | Localização em caso de wandering (demência) |

### Comunicação Pet-Idoso

Para idosos com demência ou afasia:
- Pet como **âncora emocional** quando palavras falham
- Tradução de vocalizações como **ponte comunicativa**
- Redução de agitação via interação mediada por IA

## Pessoas com Deficiência (PCDs)

### Deficiência Visual
- **Guia cão + CIH**: Descrição audível do que o cão "está dizendo"
- Alertas de obstáculos via vibração + voz

### Deficiência Auditiva
- **Tradução visual**: Display mostrando o que o cão está comunicando
- Alertas luminosos para sons importantes (campainha, alarme)

### Autismo e TEA
- Pets como **co-reguladores emocionais**
- Tradução de estados de ansiedade do pet (e vice-versa)
- Rotinas estruturadas via lembretes do sistema

## Sistema de Alerta para Cuidadores

```
Coleira detecta:
  ├── Queda do idoso (acelerômetro + padrão)
  ├── Pet em distress (vocalização anômala)
  ├── Ausência de movimento por 12h+
  └── Temperatura ambiente extrema
        ↓
   Notificação push para cuidador/família
   + Registro em log médico
```

## Benefícios Comprovados

Estudos mostram:
- **Redução de 30%** em episódios de ansiedade em idosos com pets
- **Aumento de 45%** em atividade física com companheiros animais
- **Detecção precoce** de doenças via mudanças em padrões de vocalização
