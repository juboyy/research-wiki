# O Projeto AMIAU

<span class="audience-badge badge-iniciante">🟢 Iniciante</span>

## Missão

Criar infraestrutura tecnológica de comunicação interespécies baseada em evidências científicas, com aplicações mensuráveis em bem-estar animal, segurança pública e saúde assistiva.

## Visão

Em 2030, a comunicação mediada por IA entre humanos e animais será tão comum quanto o GPS hoje. Não por crença, mas porque os dados a suportam.

## Valores

| Valor | Prática |
|-------|---------|
| Evidência antes de hype | Todo artigo precisa de fonte verificável |
| Transparência radical | Código aberto, dados abertos, limitações declaradas |
| Ética como engenharia | Consentimento, privacidade e bem-estar são requisitos, não afterthoughts |
| Interdisciplinaridade | Neurociência, engenharia, design e legislação trabalham juntas |

## Como Contribuir

### Para Pesquisadores
- Submeta correções ou adições via [issue no GitHub](https://github.com/juboyy/research-wiki/issues)
- Envie papers relevantes para inclusão na revisão sistemática
- Colabore em análise de qualidade (checklist PRISMA)

### Para Engenheiros
- Firmware: repositório `amiau-firmware` (em breve)
- API: documentação em `/docs/api-reference`
- Hardware: especificações em `/docs/hardware`

### Para Leitores
- Compartilhe artigos que achou úteis
- Reporte erros ou links quebrados
- Sugira temas para próximos artigos do AutoResearchClaw

### Para Investidores
- Dados de mercado: `/docs/mercado-pet-tech`, `/docs/mercado-seguranca`, `/docs/mercado-acessibilidade`
- Pitch deck: disponível sob NDA — contato via issue
