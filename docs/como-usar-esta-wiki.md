# Como Usar Esta Wiki

Esta wiki foi projetada para **múltiplos públicos** com necessidades diferentes. Escolha seu perfil:

## 🎓 Sou Acadêmico/Pesquisador

**O que encontrar:**
- Revisões sistemáticas com metodologia PRISMA
- Papers com hipóteses testáveis e dados primários
- Arquiteturas técnicas documentadas (CIH)
- Referências com DOI e fontes primárias

**Onde começar:**
1. [Consciousness Interface Harness](/docs/consciousness-interface-harness)
2. [Metodologia AutoResearch](/docs/metodologia-autoresearch)
3. [Revisão Sistemática: Consciência Animal](/docs/revisao-sistematica-consciencia)

**Qualidade garantida:**
- Mínimo 15 referências por artigo
- 70% fontes primárias (peer-reviewed)
- Abstract estruturado (Background, Methods, Results, Conclusions)
- Dados quantitativos com intervalos de confiança quando aplicável

## 💻 Sou Engenheiro/Técnico

**O que encontrar:**
- Especificações de hardware (diagramas, pinouts, consumo)
- Documentação de APIs com exemplos de código
- Protocolos de comunicação (MQTT, LoRa, BLE)
- Arquiteturas de deployment e DevOps

**Onde começar:**
1. [Hardware](/docs/hardware)
2. [Software](/docs/software)
3. [API Reference](/docs/api-reference)
4. [Protocolos de Comunicação](/docs/protocolos-comunicacao)

**Padrões:**
- Diagramas em Mermaid
- Código com syntax highlighting
- Especificações de versão
- Test cases documentados

## 📈 Sou Investidor/Analista

**O que encontrar:**
- Análises de mercado com dados de Grand View Research, Mordor Intelligence
- TAM/SAM/SOM de nichos específicos
- Mapeamento de startups com funding rounds
- Tendências regulatórias por jurisdição

**Onde começar:**
1. [Mercado Pet Tech](/docs/mercado-pet-tech)
2. [Mercado Segurança](/docs/mercado-seguranca)
3. [Mercado Acessibilidade](/docs/mercado-acessibilidade)
4. [Startups e Investimentos](/docs/startups-e-investimentos)

**Fontes:**
- PitchBook, Crunchbase para funding
- Relatórios setoriais anuais
- Dados de patentes (USPTO, INPI, EPO)
- Projeções com CAGR 2024-2030

## 🐕 Sou Tutor/Cuidador/Protetor de Animais

**O que encontrar:**
- Guias práticos de uso de wearables
- Interpretação de dados de atividade/saúde
- Direitos e legislação animal por país
- Comunidades e recursos

**Onde começar:**
1. [Conexão](/docs/conexao)
2. [Glossário](/docs/glossario)
3. Blog da Revista (casos reais)

## 🧠 Curioso sobre Consciência/IA

**O que encontrar:**
- Sínteses acessíveis de papers complexos
- Histórias de animais extraordinários
- Filosofia da mente sem jargon
- Podcasts e documentários recomendados

**Onde começar:**
1. [Introdução](/docs/intro)

## Navegação Transversal

Nossos artigos usam **tags semânticas** que cruzam públicos:

| Tag | Significado |
|-----|-------------|
| `#paper` | Artigo com rigor acadêmico |
| `#tech-spec` | Especificação técnica |
| `#market-analysis` | Dados de mercado |
| `#case-study` | Caso real documentado |
| `#opinion` | Editorial/posicionamento |
| `#tutorial` | Guia prático passo a passo |

## Níveis de Profundidade

Cada artigo indica seu nível:
- 🟢 **Introdução** — Acessível a qualquer público
- 🟡 **Intermediário** — Requer familiaridade básica com o tema
- 🔴 **Avançado** — Requer background técnico ou acadêmico
